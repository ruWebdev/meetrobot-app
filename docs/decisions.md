# Decisions

## Why NestJS
- Provides a robust, modular structure for scalable backend development.
- Strong TypeScript support and dependency injection.

## Why Prisma
- Type-safe database access.
- Easy schema management and migrations.

## Why Telegram-first
- Eliminates the need for traditional authentication (passwords/email).
- Provides a unified platform for both bot-driven interactions and web app UI.
- High accessibility for users within their primary messenger.
